Monatsablauf

  • Der Betreute legt die Dienstzeiten für jeden Tag fest.
  • Der Betreute bestätigt seine Eingabe, daraufhin wird eine Mail an das Team geschickt mit der Bitte mögliche Termine einzutragen. Hier ist ein Link enthalten.
  • Alle Assistenten tragen sich ein.
  • Der Betreute bekommt einen Dienstplan-Vorschlag, den er noch editieren kann.
  • Der Betreute “finalisiert” den Dienstplan - es wird eine Mail an alle mit dem Link auf den Dienstplan geschickt.

User-Managment

Neue Team-Mitglieder können nur von Patrick angelegt werden. Das Passwort ist Standardmäßig "Hallo123", wobei der Benutzer beim ersten Login sein Passwort ändern muss.

ToDo Manager

Codierung der Wiederholungen: 1;d;c
erster + zweiter Parameter: Wiederholungsinterval 1;d = every day
d = day, w = week, m = month, y = year;
dritter Paramter: d = repeat from due date, c = repeat from completion date

Gemischtes

  • Bei der Dienstplanerstellung auch auf die letzten Dienste des Vormonates achten.
  • Dienste sollten nach Möglichkeit "blockweise" (das heißt mehrere Tage hintereinander) sein.
  • Die Vorlieben des Betreuten sollten vor der Ausschöpfung des Stundenkontigentes priorisiert werden.
  • Eine Bereitschaftsstunde wird mit einer halben Stunde angerechnet.
